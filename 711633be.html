常用工具 Devops Docker Linux 云计算 2023/10/27

一、安装jenkins

条件：

安装docker服务平台的服务器

参考：https://www.runoob.com/docker/centos-docker-install.html（根据自己的服务器平台选择）

安装命令：

docker pull jenkins/jenkins:2.426-jdk21  #获取jenkins镜像
docker run -itd --name jenkins --restart=always -p 8080:8080 -p 50000:50000 -v /var/jenkins:/var/jenkins_home -v /etc/localtime:/etc/localtime  jenkins/jenkins:2.426-jdk21  #启动容器
chmod  -R 777  /var/jenkins/    #对jenkins目录进行授权
docker restart jenkins  #重新启动jenkins容器
docker ps -a  #查看jenkins服务状态

初始化jenkins：

通过浏览器访问：http://服务器ip:8080

[root@master ~]# cat /var/jenkins/secrets/initialAdminPassword
7c03d06799a64580b559fdce38299e6c
